# 📱 Product Catalog App (React Native)

## 🚀 Description
This is a mobile application built using React Native.  
The app allows users to browse products from an online API and save their favorite items locally using SQLite.

---

## ✨ Features

-  Fetch products from an API (FakeStore API)
-  Display products with image, title, and price
-  Save products locally using SQLite
-  Offline access to saved products
-  Delete saved products

---

## 🛠️ Technologies Used

- React Native (Expo)
- SQLite (expo-sqlite)
- JavaScript (ES6)
- FlatList
- Fetch API

## ⚙️ Installation

1. Clone the repository:
```
git clone https://github.com/USERNAME/Product-Catalog-App.git
```

2. Navigate to the project folder:
```
cd Product-Catalog-App
```

3. Install dependencies:
```
npm install
```

4. Start the project:
```
npx expo start
```
